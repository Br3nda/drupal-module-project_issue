-------------------------------------------------------------
New Installation
-------------------------------------------------------------

1. Install the project module (http://drupal.org/project/project),
   which the project issue tracking module requires and depends on.

2. Install the project issue tracking module 
   (http://drupal.org/project/project_issue).

3. Enable the project issue tracking module under admin/build/modules

4. Configure the status messages at admin/project/project-issue-status

5. Grant the proper access to users under admin/user/access
